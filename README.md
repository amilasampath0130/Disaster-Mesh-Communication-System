# Disaster Mesh Communication System

## Project Title
Long-Range Communication Mesh Network for Disaster Response and Relief Coordination

## Description
This project develops a long-range communication system that works even when internet or cellular networks fail during disasters. The system uses LoRa mesh networking and a mobile application to enable communication between rescue teams and affected people.

## Objectives
- Provide communication when cellular networks are unavailable
- Enable long-distance message transmission
- Build a decentralized mesh communication network
- Support disaster response coordination

## System Architecture

Mobile Application  
↓  
Bluetooth (BLE)  
↓  
ESP32 Microcontroller  
↓  
LoRa Transmission  
↓  
Mesh Network Nodes  
↓  
Receiving Node  
↓  
Mobile Application

## Technologies Used
- ESP32
- LoRa Communication
- Bluetooth Low Energy (BLE)
- React Native (Expo)
- Mesh Networking
